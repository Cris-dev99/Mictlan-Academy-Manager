&#x20;**Mictlan Academy Manager**



Sistema de gestión de alumnos desarrollado en Java y MySQL para la administración de estudiantes de una academia.



&#x20;  **Tecnologías utilizadas**



\* Java

\* MySQL

\* JDBC

\* Maven

\* Swing



&#x20; **Funcionalidades**



\* Registro de alumnos.

\* Visualización de alumnos en tabla.

\* Edición de información de alumnos.

\* Selección de registros desde la tabla para facilitar su modificación.

\* Gestión de estados de alumnos:



&#x20; \* Activo

&#x20; \* Inactivo

&#x20; \* Suspendido

&#x20; \* Vetado

\* Registro automático de fecha de inscripción.



&#x20;  **Información almacenada**



Cada alumno cuenta con los siguientes datos:



\* Nombre

\* Teléfono

\* Fecha de nacimiento

\* Estado

\* Fecha de registro (generada automáticamente por el sistema)



&#x20;  **Objetivo del proyecto**



Este proyecto fue desarrollado para poner en práctica conceptos de:



\* Programación Orientada a Objetos (POO)

\* Desarrollo Backend con Java

\* Conexión a bases de datos mediante JDBC

\* Operaciones CRUD

\* Gestión de información utilizando MySQL



&#x20;  **Mejoras futuras**



\* Búsqueda de alumnos.

\* Gestión de clases.

\* Control de asistencia.

\* Reportes de alumnos.

\* Gestión de pagos.

\* Estadísticas y métricas de inscripción.
